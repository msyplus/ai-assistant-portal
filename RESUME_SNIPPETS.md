## AI 产品运营版

**AI 服务系统作品集 | 独立开发者 | 2026.03 - 至今**

- 使用 Claude Code + Streamlit 独立构建 5 个 AI 服务运营 Demo，覆盖 VOC 智能分类、批量异常预警、客服对话质检、服务事件摘要和统一门户。
- 按“分类 → 风险识别 → 质量评估 → 摘要 → 入口整合”的递进逻辑设计产品链路，复现并扩展过往在携程 AI 智能客服与预警平台中的业务经验。
- 采用“规则/统计引擎兜底 + 多模型 AI 增强”的架构，保证无 API Key 场景下仍可演示核心流程，接入 DeepSeek、Gemini、Groq 或本地 Ollama 后可增强语义分析能力。
- 通过结构化输出、Badcase 反馈、人工评分和报告导出，展示将业务规则、评估标准和 Prompt 约束转化为可运行系统的能力。
